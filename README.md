# rss-feeds

本项目的目的旨在收集比较优秀的 rss feed，提供给大家订阅使用。

如果你有想添加的新 rss feed，可以提 PR。或者发邮件给我（mailto: setriones@gmail.com）

提交 rss feed 请带下面三个信息：

* rss feed 的名字
* rss feed 的链接
* rss feed 的介绍

| 名字 | 链接 | 介绍 |
| ---- | ---- | ---- |
